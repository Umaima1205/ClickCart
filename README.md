# ClickCart

An Ecommerce App built with flutter

# This app includes:
- A Welcome Slider
- Home Page
- Bottom Navigtaion Bar
- Products Page with Categories Filters
- Products Detail Page
- Favourite Products Functionality
- Add to Cart and Place Order Functionality
